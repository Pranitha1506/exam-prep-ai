# ExamPrep AI

AI-powered exam preparation assistant built using React, Vite, TypeScript, and Tailwind CSS.

## 🚀 Features

- 📊 Dashboard with study progress
- 📝 Practice Tests
- 🧠 Flashcards
- 📚 Study Materials
- 📈 Progress Tracking
- 🤖 AI Assistant (future scope)

## 🛠️ Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS (shadcn UI)
- Radix UI

## ▶️ Run Locally

```bash
npm install
npm run dev
